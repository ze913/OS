# Elevator Scheduling System

## 简介
基于 Python + PyQt5 开发的**电梯调度模拟系统**，作为操作系统课程设计项目。系统模拟 20 层楼宇、5 部互联电梯的运行，以**多线程**模拟进程并发，完整实现 PCB、进程状态、调度算法、同步互斥与中断处理等操作系统核心机制。

## 功能特点
- 20 层、5 部电梯，初始停靠在 1 层
- 电梯独立线程运行，互不干扰
- 支持内呼、外呼、全局按钮联动
- SCAN 电梯算法 + 三级全局调度
- 数码管楼层显示、开关门动画、实时状态展示
- 手动开关门、报警中断、临界区互斥保护
- 无请求时自动静止待机

## 技术栈
- Python 3
- PyQt5（GUI、多线程、信号槽）
- QThread、状态机、互斥锁、事件驱动

## 运行方式
1. 安装依赖
```bash
pip install pyqt5
```
2. 运行程序
```bash
python elevator.py
```

## 核心原理
- 每部电梯对应一个独立线程（进程）
- 自定义 PCB 记录进程 ID、楼层、状态、任务队列
- 进程状态：就绪、运行、阻塞、用户干预、中断
- 调度：单电梯 SCAN + 全局三级优先级调度
- 同步：临界区互斥、全局任务去重
- 中断：报警为最高优先级，抢占式处理
